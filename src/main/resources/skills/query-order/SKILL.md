# 查询奶茶订单

## Instructions

当客户要查询订单时，请按照以下步骤操作：

### Step 1: 获取查询条件
可通过以下任一方式查询：
- **订单号**（最精确）
- **手机号**
- **客户姓名**

### Step 2: 执行查询
- 调用 `queryOrder` 工具查询订单
- 如客户未提供查询条件，主动询问

### Step 3: 展示结果
- 以清晰的格式展示订单信息
- 包括订单状态、饮品详情、价格等

## Guidelines
- 保护客户隐私，不展示其他客户信息
- 如果查询不到，引导客户提供更多信息
