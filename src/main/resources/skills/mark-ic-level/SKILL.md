# Update SR Data IC Tier

## Instructions

### Step 1: Call markICLevel Tool
- Call the `markICLevel` tool
- The tool updates IC tier information for SR-containing data

### Step 2: Present Results
- Add 2 new columns on the rightmost side of the detail data Excel: Company Tier, IC Tier
- Confirm update completion

## Guidelines
- SR data requires special IC tier handling
- IC tier marks the counterparty company's tier in inter-company transactions
