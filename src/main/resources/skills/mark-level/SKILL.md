# Retrieve Equity Info and Mark 8 Major Tiers

## Instructions

### Step 1: Call markLevel Tool
- Call the `markLevel` tool to obtain equity relationship data
- The tool returns a list of companies with their tier information

### Step 2: Present Results
- Add the tier information to the existing detail data Excel
- Add a new column on the rightmost side: "Investee Company's 8-Tier Classification"
- Insert data into Sheet3 (Ownership Data)

## Guidelines
- The 8 major tiers are the classification criteria of the group's equity structure
- Each company can only belong to one tier
